# MFCenter-PaperWallet

Бумажный кошелек MFCoin.

Фичи:
* генерация бумажных кошельков;
* генерация пар privkey-pubkey;
* расшифровка адресов, полученных с vanitygen pool;
* использование brainwallet;
* шифрование приватных ключей;
